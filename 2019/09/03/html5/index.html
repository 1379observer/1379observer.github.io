背景知识

HTML：超文本标记语言（Hypertext Markup Language） W3C标准

“超文本”：指页面可以包含图片，链接，甚至音乐，程序等非文字元素。
W3C标准：
        结构 HTML
        表现 CSS
        行为 JavaScript（重点难点）

JavaScript：用来给HTML网页增加动态功能
HTML5是HTML规范的最新版本
一系列用来制作现代富Web内容的相关技术的总称
最重要的三项技术：
    HTML5核心规范
    CSS（层叠样式彪）

HTML5新特性
    引入原生多媒体支持
    引入可编程内容

块元素和内联元素

没有任何语义
主要用来做页面中的布局，内联元素主要用来选中文本设置样式

块元素：块元素 会独占一整行

内联元素（行内元素）：只占自身大小的元素

乱码问题

原因：解码编码所采用的规则--字符集（ASCII、ISO-8859-1、GBK、GB2312、UTF-8），不同
解决方法：告诉浏览器网页所采用的编码字符集，meta标签用来设计网页的一些元数据，
        比如网页的字符集，关键字，简介，且是一个自结束符

实体

特殊字符不能直接使用，需要使用一些特殊符号来表示这些特殊字符–实体（转义字符串）

图片

使用img标签来引入一个外部图片，img标签是自结束标签
属性：

src：设置一个外部图片路径
            相对路径：指对于当前资源所在目录的位置
            绝对路径：
      alt：可以用来设置图片不能显示时，对图片的描述
            收索引擎可以用过alt属性来标识不同的图片
            如果不屑alt属性，则搜索引擎不会对img进行收录
     width：改变图片的宽度，单位一般为px
     height：改变图片的高度，单位一般为px
图片格式：JPEG（JPG） 支持颜色多，不支持透明，图片可以压缩
         GIF 支持颜色少，支支持简单的透明，支持动态图
         PNG 支持颜色多，且支持复杂透明

引入一个外部的页面，使用iframe来创建一个内联框架

属性：src：指向一个外部页面路径，也可使用相对路径
width:… height:…
naem:可以为框架指定一个nmae属性

超链接

使用超链接可以让我们从一个页面跳转到另一个页面

属性：

href：指向链接跳转目标地址（相对路径或可以写一个完整的地址）
            设置为#时，超链接会跳转到当前页面顶部
            mailto：邮件地址 默认打开计算机默认电子邮件客户端，并且件收件人设置mailto后的邮箱地址
     id：标签的唯一标识
     target：指定打开链接位置
             可以设置直一个内联框架的name属性值，链接将在指定的内联框架打开

CSS语法

注释：

选择器

通过选择器可以选中页面中的指定元素，并且将申明块中的样式应用到选择器上的对应元素上
元素选择器：
    作用：选择页面中所有指定元素
    语法：标签名{}
id选择器：
    作用：通过元素id属性值选中唯一的一个元素
    语法：#id属性值{}
class选择器：
    作用：通过元素class属性值选中多个元素
    语法：.class{}

    注意：一个元素可以有多个class，多个值用空格隔开
选择器分组（并集选择器）：
    作用：可以同时选中多个选择器对应元素
    语法：选择器1，选择器2，选择器N{}
通配选择器：
    作用：选中页面所有元素
    语法：*{}
复合选择器：
    作用：选中同时满足多个选择器的元素
    语法：选择器1选择器2选择器N{}
后代选择器：
    作用：选中指定元素的后代元素
    语法：祖先元素 后代元素{}
子元素选择器：
    作用：选中指定父元素的指定字元素
    语法：父元素>子元素{}

声明块

声明块紧跟在选择器后面，用一对{}括起来，声明间用;隔开,声明的样式名和样式值用:连接

元素关系

父元素：直接包含子元素的元素
子元素：直接被父元素包含的元素
祖先元素：直接或间接包含后代元素的元素
后代元素：直接或间接被祖先元素包含的元素
兄弟元素：拥有相同父元素的元素

伪类选择器

正常链接
a:link{}
访问过链接
a:visited{}(只能定义字体颜色)
鼠标滑过的链接
a:hover{}
正在点击的链接
a:active{}(涉及用户隐私，visited所以只能设置字体颜色)
获取焦点
:focus{}(文本框获取焦点后，可以修改内容)

伪元素

来表示元素中一些特殊的位置

为P中设置第一行设置一个背景颜色

为p中第一个字符设置特殊样式

:before表示元素最前边

:after表示元素最后边

属性选择器

作用：可以根据元素中属性或属性值来选取指定元素
语法：
    [属性名]选取含有指定属性的元素
    [属性名="属性值"]
    [属性名^="属性值"]选取含有指定属性值内容开头的元素
    [属性名&="属性值"]选取含有指定属性值内容结尾的元素
    [属性名*="属性值"]选取含有指定属性值内容包含的元素

title属性：可以给任何标签指定，当鼠标移到元素上，元素中title属性值将会作为提示文字显示

其他子元素选择器

:first-child 可以选中第一个子元素
:last-child 可以选中最后一个子元素
:nth-child(number) 可以选中第number个字元素
        even表示偶数位置的子元素
        odd表示奇数位置的子元素
当前类型所有子元素中排列
:first-of-type
:last-of-type
:nth-of-type

兄弟选择器

作用：选中一个元素后挨紧着的指定兄弟元素
语法：前一个 + 后一个
作用：选中一个元素后面所有的兄弟元素
语法：前一个 ~ 后边所有

否定伪类

作用：可以从已选中的元素中剔除某些元素
语法：:not(选择器){}

样式继承

子继承父或祖的样式
但是不是所有的样式都会被继承，比如：背景相关的样式都不会被继承

选择器优先级

优先级规则：
    内联样式    1000
    id选择器    100
    类和伪类    10
    元素选择器  1
    通配*       0
    继承的样式  没有优先级
1.当选择器中包含多种选择器时，需要将多种选择器的优先级相加然后比较
2.如果选择器优先级相同，则使用靠后的样式
3.并集选择器优先级是单独计算的：
    div,p,#p1,.hello{}
4.在样式最后添加一个!important，改样式将获得最高优先级

文本格式化

单位：
    长度单位：
        像素 px
          em 和百分比类似，相对于当前元素的字体大小来计算的
          lem= 1font-size
字体：
    font-size 并不是设置字体本身大小，而是设置格的高度
    font-family 
    font-style 设置文字的斜体 
    font-weight 设置文本加粗效果
    font-variant 用来设置小型大写字母
    分类 serif(衬线字体)
        sans-serif(非衬线字体)
        monospace(等宽字体)
        cursive(草书字体)
        fantasy(虚幻字体)

    font 同时设置字体相关所有样式
         要求文字的大小和字体必须写，而且字体必须最后一个样式，大小必须倒数第二个样式
行间距：
    通过设置行高来间接设置行间距，行高越大行间距越大
    使用line-height来设置行高
    行间距 = 行高 - 字体大小
    接受值：
        1. 接受一个大小
        2.指定一个百分数，则会相对与字体去计算行高
        3.一个数值，则行高会设置字体大小相应倍数

文本样式

text-transform可以用来设置文本大小写
    可选值：
        none 默认值
        capitalize 单词首字母大写
        uppercase 所有单词都大写
        lowercase 所有单词都小写 
text-decoration用来设置文本的修饰
    可选值：
        none默认值
        underline 添加下划线
        overline 上划线
        line-through 删除线
去除超链接下划线，需将text-decoration值设为none
text-spacing 设置字符间距
word-spacing 设置词与词间距离
text-align 设置文本对齐格式
text-indent 设置首行缩进

盒子模型（框模型）

浏览器默认样式

在编写样式之前需要将浏览器中的默认样式margin和padding统统去掉

内联元素盒模型

内联元素：
    不能设置宽和高
    可以设置水平方向内边距
    可以设置垂直方向内边距，但不影响页面布局
    可以设置边框，但垂直边框不影响布局
    可以设置水平方向外边距，不会重叠
    不支持垂直外边距

改变元素

通过display样式可以修改元素的类型
    可选值：
        inline:可以将一个元素作为内联元素显示
        block:可以建一个元素设置为块元素
        inline-block:将一个元素转换为行内块元素
                    一个元素既有行内元素的特点又有块元素的特点
                    即可以设置宽高又没有独占一行
        none:不显示元素，并且元素不会在页面继续占有位置
visibility用来设置元素隐藏和显示的状态
    可选值：
        visible 默认值
        hidden 元素会隐藏，位置依然占着

overflow

子元素默认是存在于父元素的内容区中，理论上讲子元素的最大可以等于父元素内容区大小
如果子元素大小超过了父元素的内容区，则超过的大小（溢出内容）会在父元素以为的位置显示
父元素默认是将溢出内容在父元素外边显示
overflow可以设置父元素如何处理溢出内容：
    可选值：
        visible：不对溢出内容进行处理
        hidden：溢出内容被修将，不显示
        scroll：是否溢出都将会为父元素添加滚动条
        auto：会根据需要添加滚动条

文档流

处在网页最底层，表示一个页面中的位置，我们所创建的元素默认都处在文档流中
在文档流中，子元素宽度默认占父元素的全部
元素在文档流中的特点
    块元素：
            1.独占一行，自上向下排列
    内联元素
            1.占自身大小，自左向右
当元素的宽度或高度设置为auto时，此时指定内边距不会影响可见框的大小而是会修改宽度和高度，以适应内边距

浮动

使用float使元素浮动，脱离文档流
    可选值：
        none 默认值，元素在文档流中排列
        left 立即脱离文档流，向页面左侧浮动
        right 立刻脱离文档流，向页面右侧浮动
    1.元素浮动后会尽量向页面左上或右上浮动，直到遇到父元素的边框或其他浮动元素
    2.浮动元素不会超过他上边的兄弟元素，最多一边齐
    3.浮动元素不会盖住文字，文字会自动环绕在浮动元素周围
    4.块元素脱离文档流以后，宽度和高度都被内容撑开
    5.内联元素脱离文档流以后会变成块元素

高度塌陷

在文档流中，父元素的高度默认是被子元素撑开的，也就是子元素多高，父元素就多高
但是当为子元素设置浮动以后，子元素会完全脱离文档流，此时将会导致无法撑起父元素高度，导致父元素高度塌陷

解决方法：
    根据W3C标准，在页面中元素都有一个隐含的属性叫做Block Formatting Context
    简称BFC，该属性可以设置打开或关闭，默认是关闭的
    当开启元素的BFC以后，元素将会具有如下特性：
    a.父元素的垂直外边距不会和子元素重叠
    b.开启BFC的元素不会被浮动元素所覆盖
    c.开启BFC的元素可以包含浮动的子元素
    如何开启元素的BFC
        A.设置浮动元素
            会导致父元素的宽度丢失
                且导致下边元素上移
        B.设置元素绝对定位
        C.设置元素为inline-block
         导致宽度丢失
        D.将元素的overflow设置一个非visible的值
            ※推荐※将overflow设置为hidden是副作用最小的开启BFC的方式

    IE6及一下的浏览器并不支持BFC，所以使用这种方式不能兼容IE6
    在IE6中虽然没有BFC，但是具有另一个隐含的属性叫做hasLayout
    该属性作用和BFC类似，所以在IE6浏览器可以通过hasLayout来解决问题
    在IE6中，为一个元素指定一个宽度，则会默认开启hasLayout
    zoom:1;
    表示是放大的意思，后边跟着一个数值，写几就放大几倍
